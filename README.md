A minimalist learning platform for curated professional courses.

## 🛠️ Key Features
*   **User Auth:** Secure login and registration via Better-Auth.
*   **Private Courses:** Middleware-protected pages that require login.
*   **Modern UI:** Built with Next.js, Tailwind CSS, and Lucide icons.
*   **Live Feedback:** Loading spinners and Toast notifications for better UX.

## 📦 Packages Used
*   `better-auth` (Authentication)
*   `mongodb` (Database)
*   `react-hot-toast` (Notifications)
*   `lucide-react` (Icons)
*   `next` (Framework)

## 🔧 Setup
1. **Install:** `npm install`
2. **Env Setup:** Add `MONGODB_URI` and `BETTER_AUTH_SECRET` to `.env`.
3. **Run:** `npm run dev`
